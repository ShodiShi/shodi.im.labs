В ходе работы было смоделировано поступление запросов на сервер в виде простейшего пуассоновского потока. Для выбранных параметров lambda = 2, T = 10 и N = 100 было получено эмпирическое распределение числа запросов за заданный интервал времени. По результатам моделирования среднее число запросов составило 19,46, а дисперсия 19,59.

Полученные значения близки к теоретическим значениям для пуассоновского распределения: M[X] = lambda * T = 20 и D[X] = lambda * T = 20. Это говорит о том, что модель работает правильно, а результаты эксперимента согласуются с теорией. Также видно, что при не очень большом числе экспериментов эмпирическое распределение получается немного неровным, но в целом повторяет форму теоретического распределения Пуассона.
