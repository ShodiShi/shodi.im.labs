# Отчет по лабораторной работе

В ходе выполнения лабораторной работы была разработана и исследована модель смены погодных состояний на основе непрерывной цепи Маркова с тремя состояниями:

- Ясно
- Облачно
- Пасмурно

Для описания процесса использовалась матрица интенсивностей переходов `Q`, в которой недиагональные элементы задают интенсивности переходов между различными состояниями, а диагональные элементы вычисляются автоматически как отрицательная сумма остальных элементов строки. Это обеспечивает выполнение условия генератора непрерывной цепи Маркова:

`sum(Q[i, j]) = 0`

Моделирование выполнялось методом статистических испытаний. Для каждого текущего состояния случайным образом определялось:

1. время пребывания в данном состоянии;
2. следующее состояние, в которое произойдет переход.

Время до следующего перехода моделировалось по экспоненциальному распределению, а выбор нового состояния осуществлялся на основе нормированных интенсивностей переходов.

В программной реализации были выполнены следующие этапы:

- ввод интенсивностей переходов через графический интерфейс;
- автоматическое вычисление диагональных элементов матрицы `Q`;
- моделирование процесса в реальном времени с помощью таймера;
- отображение текущего состояния системы;
- построение графика изменения эмпирических частот состояний;
- накопление статистики по времени пребывания в каждом состоянии;
- сохранение результатов моделирования в файл формата `.csv`.

Для нахождения теоретического стационарного распределения вероятностей была составлена система линейных уравнений:

`pi * Q = 0`

с дополнительным условием нормировки:

`pi1 + pi2 + pi3 = 1`

Решение системы выполнялось методом Крамера.

После завершения эксперимента вычислялись:

- эмпирические вероятности состояний;
- теоретические стационарные вероятности;
- отклонение между эмпирическими и теоретическими значениями.

В ходе моделирования было установлено, что при увеличении времени наблюдения эмпирические вероятности постепенно приближаются к теоретическим значениям стационарного распределения. Это подтверждает:

- корректность построенной модели;
- правильность реализации алгоритма имитационного моделирования;
- сходимость статистических оценок к теоретическим значениям.

Все результаты моделирования, включая историю переходов, матрицу интенсивностей и итоговые вероятности, автоматически сохраняются в файл формата `.csv` для последующего анализа.

Таким образом, лабораторная работа продемонстрировала практическое применение непрерывных цепей Маркова для моделирования случайных процессов, исследования динамики состояний системы и анализа их стационарного поведения.
